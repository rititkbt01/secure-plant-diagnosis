# DATA FOLDER - Plant Disease Images (8GB Total)

## Purpose
Stores all training/testing images for 3 crops:
- Tomato (10 diseases)
- Potato (8 diseases) 
- Bell Pepper (7 diseases)

## Structure
/data/raw/              - Original downloads (60GB → Don't commit!)
/data/processed/        - Clean train/val/test splits (8GB)
/data/classes.csv       - Disease → ID mapping

## Size Targets
- Training: 18,000 images
- Validation: 2,000 images  
- Test: 2,500 PlantDoc field images
