# MOBILE APP - Flutter Cross-Platform

## Purpose
Android app for farmers:
1. Camera capture (leaf photos)
2. Image compression (<300KB)
3. Secure API calls  
4. Results display + remedies

## Screens
1. Camera View → Capture button
2. Results → Disease + confidence + remedy
3. History → Past diagnoses

## Features
- Offline remedy cache
- Low-confidence warnings (<60%)
- Network retry logic
- Farmer-friendly Nepali/English UI
